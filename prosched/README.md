# Process Scheduler Simulator

**By**: Stephen Borja, Erin Chua, Zhean Ganituen, and Aaron Go.

## How To

Simply run:

```bash
# Build project
make b

# Run project
make r

# Build then run project
make br
```

You can also do it manually via:

```bash
cmake -S . -B build
cmake --build build
./build/prosched
```
